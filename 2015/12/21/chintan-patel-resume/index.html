Portfolio

Picture Buddy

Winner at #OrlandoDevs Angular 2 Hack Night
We build a application which shows different images based on the distance data. We used Arduino board with Proximity sensor to get the distance. The NodeJS server receives the distance from the board, does minor calculated distance from millimeters (mm) to meters(m) and send to Angular 2 front-end using Socket.IO. Based on the distance of the object pointed to the proximity sensor, different images are displayed on web page.

Vote My hack

at #OrlandoDevs React Hack Night
We tried to build a simple voting application using ReactJS. This application uses React, NodeJS and Express.

Social Sentiment Analyzer - Angular 1

at NeoReach Hack
We build an application which uses MEAN stack. The application requests a twitter hash, gets the most recent 100 tweets and based on each tweet, shows the graph of how positive/negative/neutral users' emotions are. The system uses MongoLabs, Express 4, Angular 1.5, D3 graphs and NodeJS. The data is stored in MongoDB for future comparison.

Social Sentiment Analyzer - Angular 2
