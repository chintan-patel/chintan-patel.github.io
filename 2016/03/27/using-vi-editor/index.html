Using Vi editor

27 March 2016

Guide Audience

Not using Unix or Macbook Terminal because you are scared of vi or vim, this is the guide for you...!

Do not worry, this steps will make you familiar with commands.

Before we get started, download this sample file, to test this commands.

Modes in vi

There are three vi modes:

  • vi mode - This is to use cut, copy and paste commands.
    • Press ESC to return to vi mode.
  • input/insert mode
    • Press i to start insert mode
  • command mode - This is used editor settings like line numbers, file operations, etc.
    • Press : to start command mode

vi mode

Make sure that all the commands are performed in ESC mode.

Action: Cut

  • Navigate to a line
  • Press dd

Behavior: The line will disappear and saved in clipboard.

Action: Undo changes

  • Press u

Action: Redo changes

  • Press CTRL+r

Action: Copy

  • Navigate to a line
  • Press yy

Behavior: The line will disappear and saved in clipboard.

Action: Paste

  • Perform Cut/Copy action
  • Navigate to a line
  • Press pfor paste

Behavior: The line will appear in next line to the cursor.

some more vi mode commands...

Command Action
Press $ To End Of Line (EOL)
Press 0 To Start Of Line
Press :1 To jump to First line of File
Press :12 To jump to line #12 of the file
Press SHIFT+G To jump to Last line of File
Press dd To cut the line pointed by cursor
Type /search term and ENTER to search the word in file
Press w To jump the words in the line

Input / Insert Mode

Action: Edit the document

  • Always Press ESC, to clear modes and back to vi mode
  • Press i or a or o to start the insert mode

Behavior: This will behave like normal editor, use keyboard arrows to navigate between lines.

Command Mode

To Start this mode Press :<command>, following table shows the details of usages:

Command Action
:q to close the file
:w to save the file
:wq to save and close the file
:w newfile.txt to save as `newfile.txt`
:q! to close file without saving changes
:wq! to save and close file
