  • Look professional
  • Quicker results
  • No need of mouse and clicks
  • Makes you feel good

Lets get started quickly...

  1. Basics
  2. Advanced

Basics

Commands

  • hostname
  • ls
  • rm
  • wc
  • grep
  • find
  • watch
  • top
  • tree

hostname

To get the hostname of the current system.

ls

ls -ltra

Say it loud l s minus l t r a

Always use ls -ltra for listing directories, it will show all hidden as well as arrange files as per last modified date.

rm

rm -rf

Say it loud r m minus r f

Used to remove directory/files forcefully.

Must be used carefully, as it deletes all files and folders within directory without prompt

wc

wc -l

Say it loud w c minus l

It will count number of lines in a file

grep

grep "<search string>" <filename>

Say it loud grep

Grep will search the string in the file and will print the line to standard output

find

find . -type f

Say it loud find dot minus type f

  • find - command
  • . - current directory
  • -type f - Only list files alternatively use type -d to find only directories

List all the files recursively in all folders of current directory.

watch

watch <command>

Say it loud watch
This command will re-run the command every 2 secs (by default).

Its good when you want to monitor or watch for files. eg. execute this command 

mkdir new_dir  
cd new_dir  
watch "ls -tlra"`

and open another terminal window 

cd new_dir  
touch new_file

The first window will show the new file added within 2 seconds.

For OSX, use brew install watch if command does not work

top

Say it loud top

Its command line activity monitor/ task manager for terminal

Suprise yourself with brew install htop and then use htop instead.

tree

tree -L 2

Say it loud tree minus L two

It will generate a tree view for directories and files with level 2

Install via brew install tree

Advanced

Combine find and grep command

List all the files in current directory with .js file extension

find . -type f | grep ".js$"

Combine find, xargs and grep command

List all the files in current directory with 'foo' word

find . -type f | xargs grep "foo"

Combine find, xargs and grep command with exclude some folders

List all the files in current directory with 'foo' word except file path having in node_modules and bower_components folder

find . -type f | egrep -v "\/node_modules\/|\/bower_components\/" | xargs grep "foo"

Search multiple strings in list of files

find . -type f | xargs egrep "foo|bar"

Search multiple string in list of files where all patterns are in a file

Create a file with string needs to be searched for: 

$ cat lookup-patterns-file
Output:  
foo  
bar  
matching-pattern

Search for all strings matching in file lookup-patterns-file against folders

find . -type f | xargs grep -f patterns-file
