Using Vi Editor

16 February 2016

Guide Audience

Not using Unix or Macbook Terminal because you are scared of vi or vim, this is the guide for you...!

Do not worry, this steps will make you familiar with commands.

Basic understanding of vi

vi has three modes:

  • vi mode - This is to use cut, copy and paste commands.
    • Press ESC to return to vi mode.
  • input/insert mode
    • Press i to start insert mode
  • command mode - This is used editor settings like setting the line numbers, etc.
    • Press : to start command mode

vi mode

Action: Cut

  • Navigate to a line
  • Press dd

Behavior: The line will disappear and saved in clipboard.

Action: Paste

This will work only if you have performed Cut action.

  • Navigate to a line
  • Press p (pfor paste)

Behavior: The line will disappear and saved in clipboard.
